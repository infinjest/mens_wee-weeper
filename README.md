# 10X10 minesweeper inspired by Gachimuchi

## Stack

- **Backend**: Python 3.13.5
- **Frontend (GUI)**: Tkinter (built-in Python library, tk==0.1.0)
- **Graphics**: Pillow (PIL) (for drawing images for mines and flags, pillow==12.1.1)
- **Sounds**: pygame.mixer (part of Pygame, for sound effects, pygame==2.6.1)
